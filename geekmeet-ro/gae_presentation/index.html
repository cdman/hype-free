Some more goodies

  • Almost all of the components are open-source
  • Free tier!
  • Run it on your own servers!*
  • It can "just work"

*Via AppScale, a 3rd party open source project

Not a silver bullet!

  • Exposes failures of distributed systems early
  • Need to write client-side code for error handling (retry, notification)
  • Sometimes hard to debug
  • Blocked in certain countries
  • Need to be aware of quotas, deadlines, limits, ...
  • Follow your logs religiously* and fix issues found
  • Once data is in the GAE datastore, it can be hard to bulk-extract.

* Especially because they are overwritten periodically
